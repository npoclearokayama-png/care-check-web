# このチェックを使う方へ

このツールは、今の育児負担を短時間で整理するためのセルフチェックです。  
診断や判定を行うものではありません。

日々のしんどさは、はっきり言葉にしにくいことがあります。  
このチェックは、その「なんとなく苦しい」を少し整理して、今何を負担に感じているのかをセルフチェックするためのものです。

## このツールでできること

- 24問の質問に答えて、今の負担感を分析できます
- 4つの領域ごとに、どこに負荷が強いかを見ることができます
- 結果をそのままコピーして、AIに相談文の整理を手伝ってもらえます
- 個人情報は入力しません
- 回答はあなたの端末内に保存されます

## 使い方

1. `index.html` を開きます  
2. 質問に答えます  
3. 結果画面で、今の負担感の傾向を確認します  
4. 必要に応じて「結果をAI相談用にコピー」を押します  
5. ChatGPT や Gemini に貼り付けて、状況整理や相談文づくりに使えます  

## 結果を見るときの考え方

点数が高いこと自体が、何かの危険性を意味するわけではありません。  
しかし、今の生活の中で抱えている負担が大きくなっている可能性があります。

逆に、点数が低いからといって、つらさが存在しないとも限りません。  
定期的にチェックして、自分を見つめるために役立ててください。


## AIに相談するときの使い方

このツールには、児相や行政など特定の施設ではなく、AIに相談しやすい形で結果を渡すためのコピー文を入れています。

使い方の例:

- 自分の今の状況を整理したい
- 家族に伝える言い方を考えたい
- 支援者や相談先に伝える文章の下書きを作りたい
- 今日からできる小さな一歩を整理したい

ただし、AIは支援機関そのものではありません。  
生活の維持が難しい、安全面の不安がある、気持ちの落ち込みが強いなどの場合は、地域の相談機関、医療機関、行政窓口、福祉機関など、実際の支援につながってください。

## このツールについての注意

- これは診断ではありません
- 医療的判断や確定的評価を行うものではありません
- 結果だけで本人や家庭の状態を決めつけないでください
- しんどさが強いときは、一人で抱え込まないでください

## 最後に

このチェックは、今の負担を少し見える形にして、次にどうするかを考えやすくするためのものです。

結果が高くても低くても、まずは今の自分たちの生活を責める材料ではなく、整えるための材料として使ってください。
