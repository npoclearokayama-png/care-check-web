# このチェックを使う方へ

このツールは、今の子育て負担を短時間で整理するためのセルフチェックです。  
診断や判定を行うものではありません。

## このツールでできること

- 30問（6軸×各5問）の質問で、ここ1か月の負担感を整理できます
- 総合スコア（100点換算）と6軸スコアを確認できます
- 上位2軸の解説から、しんどさの中心を把握できます
- 任意の補助項目（年齢帯・人数・しんどい時間帯・自由記述）を結果補助に使えます
- 回答結果をそのままコピーして、AIで相談文の下書きができます

## 使い方

1. `index.html` を開きます  
2. 任意の補助項目を入力（未入力でも可）  
3. 30問に回答します（ここ1か月の状態）
4. 結果画面で総合点・6軸・上位2軸を確認します
5. 必要に応じて「結果をAI相談用にコピー」を使います

## 注意

- これは診断ではありません
- 医療的判断や確定的評価を行うものではありません
- しんどさが強いときは、地域の相談機関・医療機関・行政窓口など実際の支援先につながってください
